# Teacher Schedule Management System

A PHP-based web application for managing teacher schedules.

## Features
- Login/Registration
- Forgot/Reset Password
- Manage Schedules (Add, Delete, View)
- Admin Panel
- Search Functionality

## Setup
1. Clone the repository.
2. Import the database schema (if available).
3. Copy `db_conn.php.example` to `db_conn.php` and update the database credentials.
4. Host on a PHP-supported server (like XAMPP or InfinityFree).
